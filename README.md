# 🧠✨ Diagnostic Expert System

> 🚀 AI-powered disease prediction using symptoms

---

## 🌐 Overview

The **Diagnostic Expert System** is a smart AI-based application that predicts diseases based on user symptoms. It combines **Machine Learning** and **Rule-Based Logic (Prolog)** to provide intelligent and fast diagnostic suggestions.

---

## 🎯 Key Features

✨ Symptom-based disease prediction
🤖 Machine Learning classification
📜 Rule-based reasoning (Prolog)
⚡ Fast and user-friendly interface
📊 Data-driven decision making

---

## 🛠️ Tech Stack

* 🐍 Python
* 🤖 Scikit-learn
* 🌐 Flask
* 📜 Prolog
* 📊 CSV Dataset

## 🚀 Getting Started

### 🔹 Clone Repository

```

```

### 🔹 Install Dependencies

```

```

